Ursus from Benevento, De nomine

An experimental scholarly digital edition of the De nomine of Ursus from Benevento from codex Casanatensis 1086 by Paolo Monella, created within the ALIM Project, 2016 (work in progress).

What you see on this page is the JavaScript transformation of the XML/TEI source code. For more information on the Ursus Project, the full documentation and for the source code (including XML/TEI and JavaScript), see the project home page on GitHub.

Warning: I tested this page on Mozilla Firefox (recommended browser), Chrome, Safari and Edge. Please note that the linguistic layer has not been reviewed yet.

Note: Please allow about 10 seconds for the page to load (or to reload after checking/unckecking one of the checkboxes).
